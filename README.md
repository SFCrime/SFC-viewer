# SFCrimeMapping

This project is built with Yeoman.io
## Versioning

```sh
$ node -v
v0.12.2
$ npm -v
2.7.4
$ bower -v
1.4.1
```

## Setup
run this to install our dependencies
```sh
npm install -g yeoman
bower install && npm install
```


##Serve the development server with
```sh
grunt server
```

That will start a livereloading server
